* Express.js
* EJS
* MongoDB & Mongoose

* MVC Architecture
* Sessions & Cookies
* File Uploads (Multer)
* Cloudinary
* Git & GitHub

This repository serves as my learning archive and documents my progress throughout the course. It will continue to be updated as I learn new concepts and build more projects
